### Список задач
- Добавление новых задач
- Редактирование существующих задач
- Отметка выполнения задач (с диалогом подтверждения)
- Перетаскивание для изменения порядка
- Автосохранение локально на устройстве

### Список покупок
- Добавление товаров
- Быстрое удаление купленных товаров (один клик)
- Перетаскивание для изменения порядка
- Кнопка "Сбросить всё" для очистки списка
- Автосохранение локально на устройстве
