# Tasks & Shopping List - Android App

Простое и удобное Android-приложение для управления задачами и списком покупок с поддержкой перетаскивания элементов.

## Особенности

### Список задач
- Добавление новых задач
- Редактирование существующих задач
- Отметка выполнения задач (с диалогом подтверждения)
- Перетаскивание для изменения порядка
- Автосохранение в SharedPreferences

### Список покупок
- Добавление товаров
- Быстрое удаление купленных товаров (один клик)
- Перетаскивание для изменения порядка
- Кнопка "Сбросить всё" для очистки списка
- Автосохранение в SharedPreferences

### Пользовательский интерфейс
- Минималистичный дизайн
- Поддержка размытого фона элементов
- Адаптивная верстка
- Интуитивная навигация между экранами
- Визуальная обратная связь при перетаскивании

## Технологии

- **Язык:** Kotlin
- **Минимальная версия Android:** Не указана (рекомендуется API 21+)
- **Архитектура:** MVC (Activity-based)
- **Хранение данных:** SharedPreferences
- **UI:** XML Layouts, Custom Adapters
